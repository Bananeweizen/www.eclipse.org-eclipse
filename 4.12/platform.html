Views, Dialogs and Toolbar

Launch Run and Debug configurations from Quick Access From the Quick Access proposals (usually accessible with Ctrl+3 shortcut) you can now directly launch any of the Run or Debug configuration available in your workspace.

General Updates

Passing directory to launcher drive to import wizard When passing a directory to the launcher (such as eclipse . or eclipse ~/git/eclipse.platform.ui), the generic "smart" project import wizard is shown and suggest a plan to import project in the workspace.

New native implementation of ImageLoader on GTK The implementation of ImageLoader on GTK is now using native GdkPixbuf API for image loading and saving. Loading and saving images with ImageLoader on GTK now have native performance, with ~200% increase in loading time for JPEG files. ImageLoader has better GIF animation support and is able to support more image formats.
