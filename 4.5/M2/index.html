Improved compiler performance Significant performance improvement has been made to the compiler on generics-heavy code. Details on affected scenarios, fix and performance results can be found on Bug 434326
PDE
Products can specify CSS file In the Product Editor, on the Customization tab, you can now specify the name of a CSS file to be used to style the product. Your product must have the Eclipse 4 CSS support installed and you must synchronize the product with its defining plug-in to keep the extension point up to date.
