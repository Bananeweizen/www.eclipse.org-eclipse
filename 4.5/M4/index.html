Sorting working sets in Project Explorer You can now manually set the order in which one the working sets are shown in the Project Explorer.

From the Project Explorer, go to the View Menu, and click on "Select Working Sets..." entry. The Working Set dialog now shows some "Up" and "Down" button that allows to sort working sets. When clicking OK, the order of Working Sets will be used by the Project Explorer.

The default order of Working Sets is now based by order of creation.
