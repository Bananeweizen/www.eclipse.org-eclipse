Welcome to Juno, the simultaneous Eclipse release for 2012, including content from sixty two Eclipse projects. Two versions of the Eclipse platform have been included in this release: the primary 4.2 version, available from the Eclipse Project downloads page, and the legacy 3.8 version, available from the 3.x stream download page. Here are some of the more noteworthy things available in the twelvth annual release of the Eclipse project SDK.

  • New features in the Platform and Equinox
  • New features for Java developers
  • New APIs in the Platform and Equinox
  • New features for plug-in developers
