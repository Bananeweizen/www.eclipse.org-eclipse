Eclipse Project 4.6 M1 - New and Noteworthy

Here are some of the more noteworthy things available in the Neon milestone build M1 which is now available for download.
We also recommend to read the Tips and Tricks, either via Help > Tips and Tricks... or online for Platform, JDT, and PDE.

  • JDT

Platform
You can now disable the CSS engine (theming) Now you can completely disable the CSS engine through a check box in the Appearance preferences. This would cause Eclipse to be rendered without any CSS support and delegate everything to the OS in terms of theme colors, shapes etc.

Workbench logger supports debug level Previously, the workbench logger wrote a trace when calling logger#debug, even if the application was not started in debug mode. Now, log entries on the debug level are ignored, if the application is not started with the debug flag (-debug). See BR 469737.
Toolbar drag and drop when no theme engine present Now you can drag and drop toolbars while having '-cssTheme none'. Previously you were not able to do so because of a minor bug.
