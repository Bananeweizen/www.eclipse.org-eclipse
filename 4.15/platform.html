General Updates

Allow configuration of enabled SWT addons in application model Loading the addons for SWT automatically enabled all addons (Drag'n'Drop, MinMax, Split, Cleanup). Using tags (DisableDnDAddon, DisableMinMaxAddon, ...) on the application model, developers can now disable unwanted addons.
Redrawn turned off by default during collapse and expand operations in treeviewer To improve interactive performance treeviewers redraw is turned off by default during the collapse and expand operation. This significantly improves these operations compare to drawing these updates synchronously.
